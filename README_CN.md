这是一个沉浸式的教育型太阳系3D模拟程序，将太空探索的奇迹带到您的浏览器中。踏上宇宙之旅，见证行星运动的优雅芭蕾，从水星的快速轨道到海王星围绕太阳的遥远华尔兹。

> ⚡ 本项目由 **[DeepV Code](https://dvcode.deepvlab.ai/)** 编程工具构建 - 让编程变得更智能、更高效

## 🎯 教育使命

**探索 • 学习 • 发现**

本项目将复杂的天文学概念转化为互动学习体验：
- 🌍 **理解轨道力学**: 观察重力如何塑造行星的运行轨迹
- 🌙 **探索月相周期**: 观察卫星如何围绕母星球起舞
- ⭐ **研究恒星形成**: 学习星座和遥远星系的知识
- 🔬 **可视化物理**: 亲眼看见引力和天体动力学的作用
- 🚀 **激发太空探索**: 点燃对我们在宇宙中位置的好奇心

## ✨ 功能特色

### 🌌 **沉浸式3D环境**
- **自由相机导航**: 从任意角度和距离探索太空
- **真实比例距离**: 体验太阳系的真正浩瀚
- **动态光照**: 见证阳光如何照亮行星和卫星

### 🪐 **科学精确性**
- **真实轨道力学**: 精确的行星周期和轨道速度
- **真实纹理**: NASA品质的天体表面细节
- **引力物理**: 逼真的引力相互作用和力场

### 🌟 **丰富的宇宙细节**
- ☀️ **我们的恒星**: 具有真实日冕效应的动态太阳表面
- 🪐 **八大行星**: 从岩质水星到气态海王星
- 🌑 **天然卫星**: 主要卫星包括地球的月球、木星的伽利略卫星群
- ☄️ **小天体**: 小行星带、彗星和太空碎片
- ✨ **深空**: 遥远的星域和银河系背景
- 🎵 **大气音效**: 太空环境音为您带来完整沉浸感

## 🛠️ 技术栈

- **核心**: JavaScript (ES6+), HTML5, CSS3
- **渲染**: WebGL / Three.js
- **构建工具**: Node.js / Vercel
- **物理引擎**: 自定义轨道力学系统

## 🚀 快速开始

### 环境要求

- [Node.js](https://nodejs.org/) (推荐v14或更高版本)
- 支持WebGL的现代浏览器

### 安装步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/KKirikaze/SolarSystem3D.git
   cd SolarSystem3D
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动您的宇宙之旅**
   ```bash
   npm start
   ```
   *打开浏览器访问 `http://localhost:3000` 开始探索！*

### 🎮 操作控制

- **鼠标**: 点击拖拽可环绕太阳系轨道观察
- **滚轮**: 缩放以探索不同尺度
- **键盘**: 额外的导航控制实现精确移动
